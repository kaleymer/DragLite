Project Overview

DragLite is a vehicle acceleration performance tracker built using an ESP32, GPS module, and accelerometer, displayed on an OLED screen. This documentation will guide you through the project's features, components, installation steps, and usage instructions.

Components

  • ESP32 Microcontroller
  • GPS Module
  • Accelerometer
  • OLED Screen
  • Connecting Wires and Breadboard
  • Power Supply

Installation
